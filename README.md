# OSlib
C library for detecting OS using preprocessor.

# Install
On linux:
* ```git clone https://github.com/rxtthin/oslib.git```
* ```cd oslib```
* ```make install```

# Supported OS's

* Windows - OS\_WINDOWS  
* Linux - OS\_LINUX  
* Mac OS - OS\_MACOS  
* Android - OS\_ANDROID
* UNIX - OS\_UNIX  
* FreeBSD - OS\_FREEBSD  
* DragonFly BSD - OS\_DRAGONFLY  
* OpenBSD - OS\_OPENBSD  
* NetBSD - OS\_NETBSD
